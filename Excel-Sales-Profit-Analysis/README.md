## Project Overview
This project analyzes an eCommerce sales dataset using Microsoft Excel to
identify sales trends, customer performance, and category-level insights.
The goal is to support data-driven business decisions through clear KPIs
and interactive dashboards.

## Dataset
The dataset contains transactional eCommerce data with the following columns:
- Order ID, Order Date, Ship Date, Ship Mode
- Customer ID, Customer Name, Segment
- Region, State, City, Postal Code
- Product ID, Category, Sub-Category, Product Name
- Sales, Quantity, Discount, Profit

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers & Filters
- Conditional Formatting
- Basic Dashboard Design

## Key KPIs
- Total Revenue
- Number of Orders
- Average Order Value
- Category-wise Order Count

## Analysis Performed
- Monthly sales trend analysis
- Sales by category and sub-category
- Top customers by revenue
- Category-wise order volume
- Regional and state-level filtering using slicers

## Dashboard Features
- KPI summary cards
- Monthly sales trend chart
- Sales by category visualization
- Interactive slicers for State, City, and Quarter

## Business Insights
- Office Supplies generate the highest order volume
- Technology contributes higher profit compared to Furniture
- Sales peak during specific months indicating seasonal demand
- A small set of customers contribute significantly to total revenue

## Limitations & Future Enhancements
- Dashboard design can be further refined
- Profitability analysis by discount levels can be added
- Planned enhancements using Power BI and Tableau
